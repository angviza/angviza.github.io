

vim /lib/systemd/system/xx.service
```
[Unit]
Description=dflc start up
After=network.target remote-fs.target nss-lookup.target

[Service]
Type=forking
ExecStart=/user/jar/xx-start.sh
KillSignal=SIGQUIT
TimeoutStopSec=5
KillMode=process
PrivateTmp=true

[Install]
WantedBy=multi-user.target
```

```sh
#3、设置开机启动

#4、取消开机启动

#5、启动项目
systemctl start xx
#5、查看启动状态日志
journalctl -xe
#6、停止运行的项目
systemctl kill xx（因为上面只配置了killsignal）
```
